# ieee-cover

Use this skill for IEEE journal cover letters, submission letters, editor-facing significance
statements, and target-journal fit framing.

Typical requests:

- "Write a cover letter for TWC."
- "Frame this manuscript for a JSAC special issue."
- "帮我写 IEEE 投稿信。"
- "Make the editor-facing significance statement less exaggerated."

Use `ieee-response` instead for point-by-point reviewer replies.

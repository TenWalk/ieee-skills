# ieee-search

Use this skill for IEEE communications literature search, DOI verification, BibTeX preparation,
related-work matrices, and novelty/citation-gap audits.

Typical requests:

- "Find recent TWC papers on RIS beamforming with imperfect CSI."
- "Verify these DOIs and make BibTeX."
- "查一下这个方向的 IEEE 相关工作。"
- "Can I safely claim novelty against prior ISAC rate-CRB papers?"

Use `ieee-citation` after verification to produce final IEEE-formatted references.

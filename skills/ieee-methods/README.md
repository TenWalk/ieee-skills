# ieee-methods

Use this skill for IEEE communications methods details that must be reproducible: notation,
parameter tables, algorithm boxes, convergence statements, and complexity analysis.

Typical requests:

- "Audit my Methods section for reproducibility."
- "Make a notation table for this system model."
- "Write the complexity analysis for my AO/SCA algorithm."
- "检查这段方法部分是否能复现。"

Load `SKILL.md` first; open the reference files only when the routing table calls for them.

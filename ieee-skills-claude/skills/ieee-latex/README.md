# ieee-latex

Use this skill to compile and diagnose IEEEtran manuscripts: BibTeX, undefined references,
overfull boxes, figure/table placement, Index Terms, EDICS, and final PDF checks.

Typical requests:

- "Compile my IEEEtran paper and fix the errors."
- "Why are my references showing as question marks?"
- "Fix overfull hboxes in this IEEE paper."
- "IEEE LaTeX 编译失败，帮我查。"

Use `ieee-citation` for reference formatting and `ieee-figure` for plot generation.
